# Diabetes Diet Assistant

A demo web app to assist diabetic individuals in making healthier food choices through a simple and user-friendly interface.

## Features

- Simple, user-friendly interface for diet guidance
- Helps diabetic users make healthier food choices
- Web-based application accessible from any browser

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Vanilla.js
- **Backend:** Node.js, Express.js
- **Database:** SQL

## Prerequisites

- Node.js and npm installed
- A SQL database (e.g., MySQL/SQLite)

## Setup & Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Dhanya-ghub/Diabetes-Diet-Assistant.git
   cd Diabetes-Diet-Assistant
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Set up the database and update connection details in the configuration file
4. Start the server
   ```bash
   node app.js
   ```
5. Open your browser and go to `http://localhost:3000` (or the configured port)

## Usage

1. Launch the app in your browser
2. Browse food choices and dietary recommendations
3. Use the interface to plan healthier meals based on diabetic-friendly options
